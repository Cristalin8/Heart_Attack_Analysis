Folderul Code contine codul integral pentru lucrarea data(ploturi, AED, crearea si compararea modelelor)
Folderul Data contine setul de date asupra caruia s-a facut analiza. El contine urmatoarele variabile:
Age : Varsta pacientului

Sex : Sexul pacientului

cp : Tipul de durere toracică 
Value 1: angină pectorală tipică
Value 2: angină atipică
Value 3: durere non-anginoasă
Value 4: asimptomatic

trtbps : numarul de batai a inimii maxim per minut

chol : colesterol în mg/dl preluat prin intermediul senzorului BMI

fbs : (glicemie > 120 mg/dl) (1 = adevărat; 0 = fals)

rest_ecg : rezultate electrocardiografice în repaus

Value 0: normal
Value 1: prezintă anomalii ale undei ST-T (inversiuni ale undei T și/sau supradenivelare sau depresie ST de > 0,05 mV)
Value 2: prezintă hipertrofie ventriculară stângă probabilă sau certă după criteriile lui Estes: frecvența cardiacă maximă obținută

thalachh : tensiune arterială în repaus (în mm Hg)

target : 0= nu a avut atac de cord 1= a avut atac de cord

Folderul Figure contine toate figurile obtinute in urma analizei acestui set de date

Folderul Raport contine articolul stiintific efectuat in urma analizei acestui set de date, cat si o prezentare PowerPoint.
